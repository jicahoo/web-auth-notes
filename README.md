## Basic Access Authentication
* 简单访问认证协议
* https://en.wikipedia.org/wiki/Basic_access_authentication
* Base64编码用户名和密码，结合HTTPS.
* 缺点: 一直要将凭证在网络上传输；将凭证存在了浏览器的缓存里；很容易找出你的密码；
